Terminal Multiplexer untuk Pengembang

TMUX adalah terminal multiplexer yang memungkinkan Anda menjalankan beberapa sesi terminal dalam satu jendela. Website ini akan membantu Anda mempelajari dasar-dasar penggunaan TMUX.

Instalasi

Ubuntu/Debian

  • sudo apt update
  • sudo apt install tmux

CentOS/RHEL/Fedora

  • sudo yum install tmux
  • # atau
  • sudo dnf install tmux

macOS

  • brew install tmux
  • # atau
  • port install tmux

Dasar-Dasar TMUX

Memulai Sesion

  • tmux # memulai sesi baru
  • tmux new -s nama_sesi # sesi dengan nama
  • tmux attach -t nama_sesi # masuk ke sesi yang ada

Keluar dari TMUX

  • exit # keluar dari sesi
  • Ctrl + b kemudian d # detach dari sesi

Manajemen Window

  • Ctrl + b kemudian c # buat window baru
  • Ctrl + b kemudian n # pindah ke window berikutnya
  • Ctrl + b kemudian p # pindah ke window sebelumnya

Manajemen Panel

Membagi Panel

  • Ctrl + b kemudian % # bagi secara vertikal
  • Ctrl + b kemudian " # bagi secara horizontal

Navigasi Panel

  • Ctrl + b kemudian arrow # pindah ke panel
  • Ctrl + b kemudian q # tampilkan nomor panel
  • Ctrl + b kemudian o # pindah ke panel berikutnya

Mengatur Ukuran Panel

  • Ctrl + b kemudian Ctrl + arrow # ubah ukuran panel
  • Ctrl + b kemudian z # zoom/unzoom panel

Tips Lanjutan

Manajemen Sesion

  • tmux ls # lihat daftar sesi
  • tmux kill-session -t nama_sesi # hapus sesi
  • tmux rename-session -t nama_lama nama_baru # ganti nama sesi

Scroll dan Copy-Paste

  • Ctrl + b kemudian [ # masuk mode copy
  • Gunakan arrow untuk navigasi
  • Space untuk mulai seleksi
  • Enter untuk copy
  • Ctrl + b kemudian ] # paste

Konfigurasi

  • Buat file ~/.tmux.conf
  • # Contoh: ubah prefix key
  • set-option -g prefix C-a
  • unbind-key C-b
  • bind-key C-a send-prefix
